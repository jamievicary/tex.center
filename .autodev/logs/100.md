## Goal for this iteration

N=100 → refactor iteration (§3). Single highest-leverage
duplication in the tree: the PGlite test harness. Three Node
scripts (`packages/db/test/{migrations,projects,users-sessions}-pglite.test.mjs`)
each redefine the same `pgliteDriver(pg)` closure (~20 lines).
Three Python wrappers (`tests_gold/cases/test_pglite_*.py`) each
redefine the same `unittest.TestCase` shelling out to one of
those scripts (~25 lines each, differing only in class name and
script path).

Extract `pgliteDriver` (plus a `freshMigratedPglite()` convenience)
into `packages/db/test/_pgliteHarness.mjs`; collapse the three
gold wrappers into one parameterised `test_pglite.py`.
