## Goal for this iteration

Land M9.gold-restructure per `.autodev/discussion/196_answer.md`:
extend `sharedLiveProject` with a one-shot warm-up that drives the
freshly-created project's Fly Machine + sidecar + daemon all the
way to an initial `pdf-segment` frame, then shrink per-spec polls
on GT-1..5. Today GT-3 and GT-5 are RED on live and each sits out
~5 min of internal timeouts on every run; the gold suite has
regressed from ~2 min to ~10–13 min as a result. Decoupling the
cold-start absorber from the per-spec assertions turns those
RED specs into ~10s fast-fails and makes every spec timeout
unambiguously a feature signal.

## Approach

Pure gold-test restructure, no production code touched.

1. `tests_gold/playwright/fixtures/sharedLiveProject.ts` — after
   `createProject` and before `use(project)`, run
   `warmUpProject(...)`. The helper mints a session against the
   worker-scoped `db` fixture (same path `authedPage` takes),
   opens a `BrowserContext` with the `tc_session` cookie set via
   `buildSessionCookieSpec`, navigates to `/editor/<id>`, and
   waits up to 180_000ms for a `payload[0] === TAG_PDF_SEGMENT`
   frame on the per-project WS. Elapsed-ms logged to stderr as
   the cold-start health metric called for in `196_answer.md` §3.
   Context closed and the warm-up session row deleted on exit.
2. Per-spec polls shrunk:
   - GT-1: `cmContent.waitFor({ state: "attached" })` 120s → 10s;
     `test.setTimeout` 300s → 60s.
   - GT-2: pdf-segment poll 240s → 5s; `test.setTimeout` 300s → 60s.
     Re-purposed as the fresh-page hydrate re-check.
   - GT-3: pdf-segment poll 240s → 5s; `cmContent` visible 30s →
     10s; post-edit poll 60s → 10s; `test.setTimeout` 360s → 60s.
   - GT-4: pdf-segment poll 240s → 5s; `cmContent` visible 30s →
     10s; `test.setTimeout` 420s → 120s (kept generous because the
     spec includes a 30s post-typing settle window by design).
   - GT-5: pdf-segment poll 240s → 5s; `cmContent` visible 30s →
     10s; `expectPreviewCanvasChanged` 60s → 10s; `test.setTimeout`
     420s → 90s.
