```
{"level":40,"projectId":"12ddd34d-...","elapsedMs":36,
 "sourceLen":275,"targetPage":1,

   (no usable rollback target for this edit)",
```

`elapsedMs=36` is the daemon completing its `recompile` round
fast (no real TeX work) and emitting `round-done` with zero
`[N.out]` shipout events. That is the upstream-supertex no-op
shape iter 188 first documented and iter 228 hypothesised here.

Conclusion: GT-5's recurrence on the polluted warm-doc body-edit
path is **upstream supertex**, the same class of bug closed by
upstream iters 755–758 for the GT-8 cold-start `\newpage` shape,
but for a different input pattern (warm doc + edit at body
offset past every extant checkpoint).
